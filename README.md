
# Supply Chain Analysis Dashboard

A comprehensive data analysis project focused on optimizing supply chain operations. This dashboard provides actionable insights into inventory management, logistics, and supplier performance using data-driven visualizations.

## 🚀 Project Overview
The objective of this analysis is to identify bottlenecks in the supply chain, evaluate lead times, and optimize stock levels. By analyzing historical data, this project helps in making informed decisions to reduce costs and improve delivery efficiency.

## ✨ Key Features
- **Inventory Optimization:** Analysis of stock levels to prevent overstocking or stockouts.
- **Logistics Performance:** Tracking of shipping times and carrier efficiency.
- **Cost Analysis:** Breakdown of manufacturing and shipping costs across different regions.
- **Supplier Evaluation:** Metrics to assess supplier reliability and lead times.
- **Interactive Visuals:** Dynamic charts that allow for deep-diving into specific product categories or timeframes.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Environment:** Jupyter Notebook
- **Dataset:** Industrial supply chain records

## 📊 Analysis Workflow
1. **Data Cleaning:** Handling missing values and formatting data types.
2. **EDA (Exploratory Data Analysis):** Identifying trends, correlations, and outliers.
3. **KPI Calculation:** Measuring metrics like Order Cycle Time and Inventory Turnover.
4. **Visualization:** Creating a final dashboard to communicate findings effectively.
